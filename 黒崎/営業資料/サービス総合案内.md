# AI導入・活用 総合サービスのご案内

---

## 私たちのAI導入の考え方

**「とりあえずAI入れましょう」はやりません。**

すべてのサービスは、貴社の**経営目標**から逆算して設計します。

1. **経営目標のヒアリング** — 売上・コスト・生産性、何を改善したいのか
2. **目標設定** — AIで解決すべき課題と、達成すべき数値目標を明確化
3. **導入設計** — 目標に最適なサービス（研修・伴走支援・インフラ整備）をご提案

「AIを入れること」がゴールではなく、**「経営課題を解決すること」がゴール**です。
その前提で、コンサルティングをさせていただきます。

## サービス① AI研修

### こんな企業に

- 社員のAIリテラシーにバラつきがある
- 「ChatGPTって何？」という社員がまだいる
- 全社員に最低限のAI知識を持たせたい

### 提供内容

- **AI入門〜応用まで、必要なコースをすべて揃えた研修パッケージ**
  - AI基礎・ツール活用・プロンプト技術・業務活用・リスク管理など幅広くカバー
  - 貴社の業種・職種に合わせてカリキュラムをカスタマイズ
- **eラーニング**：1コース約10分、すき間時間で受講可能
- **LMS（学習管理システム）完備**：受講ログ・修了証・履歴CSV自動出力
- **テスト付き**：各コース修了テストあり（やりっぱなし防止）
- オプション：集合研修・双方向オンライン研修も対応

### 助成金で研修費用の60%が返ってきます

厚労省の**定額制訓練（人への投資促進コース）**を活用すれば、研修費用の**60%が助成**されます。

| 項目 | 内容 |
|---|---|
| **助成率（中小企業）** | **60%** |
| **1人あたり上限** | 月額2万円（年間24万円） |
| **対象企業** | 全国・全規模 |
| **研修時間要件** | 対象者全員の合計10時間以上 |
| **申請回数** | 年度内3回まで（4か月ごとの分割申請OK） |
| **制度終了** | **2027年3月末** |

#### 金額シミュレーション（中小企業・10名の場合）

```
年間利用料: 1人あたり40万円 × 10名 = 400万円
助成額（60%）: 24万円 × 10名 = 240万円
→ 1人あたり実質負担: 16万円
→ 4か月ごとに分割申請できるので、立替負担も軽減
```

#### 助成金申請サポート付き

- 計画届の記載例・必要書類ガイドを提供
- LMSで助成金申請に必要な受講ログ・修了証・履歴CSVをすべて出力可能
- 社労士との連携体制あり

> **2027年3月末で制度終了予定。計画届は訓練開始の1か月前が厳守。今動けば間に合います。**

### 料金

**年間40万円/人**（助成金活用で **実質16万円/人**）

### 導入の流れ

1. ヒアリング（対象者・目的の確認）
2. コース選定・カリキュラム設計
3. LMSアカウント発行・受講開始
4. 助成金申請サポート（計画届〜支給申請まで伴走）

## サービス② AI伴走支援

### こんな企業に

- 研修はやったけど現場で活用が進まない
- AI分かる人がいない・相談相手がほしい
- AI推進できる人材を現場に入れたい

### 2つのプラン

#### AI顧問プラン

- 「AIでこれできる？」にすぐ答えるプロが顧問としてつきます
- 定期MTGで課題整理・活用アドバイス・最新情報の共有
- 部門別のAI活用ユースケース提案
- プロンプトテンプレートの作成・提供

**月額10万円**

#### AI人材派遣プラン

- AI活用スキルを持つ人材を常駐 or 業務委託で御社に派遣
- 業務内容の例：
  - 社内AI活用の推進・旗振り
  - 業務プロセスの分析 → AI化の設計・実行
  - 社内向けプロンプト・マニュアルの整備
  - 社員への実地トレーニング（OJT）
- ノウハウを社内に移管し、自走できる状態をつくります

**月額30万円**

### どっちを選ぶ？

| | AI顧問 | AI人材派遣 |
|---|---|---|
| 関わり方 | 相談・アドバイス | 現場に入って実行 |
| 頻度 | 月1〜2回のMTG | 週2〜5日の常駐 |
| 向いてる企業 | 自社で動ける人はいるが方向性に迷う | そもそも動ける人がいない |
| 料金 | **月額10万円** | **月額30万円** |

### 導入の流れ

1. ヒアリング（現状・課題・ゴールの確認）
2. プランのご提案（顧問 or 人材派遣）
3. 支援スタート
4. 月次レポート・ノウハウ移管

## サービス③ AIインフラ整備

### こんな企業に

- 「AIを導入したい」が、何から手をつけていいか分からない
- ChatGPTのアカウントは作ったが、それ以上進んでいない
- 社員が日常業務でAIを当たり前に使える環境を作りたい

### 提供内容

**Claude Code・Cursor などの最新AIツールを基盤に、社員全員がAIを使える環境を構築します。**

- **AI開発環境の構築**：Claude Code / Cursor / GitHub Copilot など、業務に最適なAIツールを選定・導入・設定
- **業務フローへのAI組み込み**：既存の業務プロセスにAIを自然に組み込む仕組みを設計
- **社内AIガイドライン策定**：利用ルール・セキュリティポリシー・データ取り扱い規定を整備
- **プロンプト・ワークフロー整備**：部門別・業務別のテンプレート、自動化フローを納品
- **運用サポート**：導入後のトラブルシューティング・アップデート対応

### 導入の流れ

1. 現状診断
2. AI導入計画の策定・ご提案
3. ツール導入・ガイドライン整備
4. 社内展開・研修実施
