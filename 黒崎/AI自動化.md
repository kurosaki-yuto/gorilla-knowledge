# AI教育講座 動画自動生成パイプライン 設計仕様書

## 概要

AI教育のオンライン講座（サブスクリプション型）の動画コンテンツを、完全自動で生成するパイプラインシステム。

## 設計思想

### 目的
- AI教育講座の動画を、人の手を一切介さず完全自動で量産する
- 対象ツール: ChatGPT / Claude 等の生成AI、Midjourney 等の画像生成AI

### コンテンツ形式
- 研修形式の構成（概念説明 + 実演）
- 実演パートは**実際のUI画面を動かしている映像**（スクリーンショットのスライドショーではない）
- リアルタイム性は不要（事前録画 → 後処理で完成させる）

### 絶対条件
- 全ステップ完全自動化（人の確認・操作なし）
- Step4の出力精査もAIが自動で行う

---

## パイプライン（5ステップ）

### Step 1: 台本生成
- **入力**: 講座テーマ（例: 「ChatGPTでメール文面を作成する方法」）
- **処理**: Claude APIでテキストベースの講座台本を自動生成
- **出力**: 台本テキスト（JSON or Markdown）
  - 講座タイトル
  - 学習目標
  - 概念説明パートの原稿
  - 操作手順（Playwrightスクリプト生成用）
  - ナレーション原稿

### Step 2: 研修スライド生成
- **入力**: Step1の台本テキスト
- **処理**: 台本から研修形式のスライドを自動生成
- **出力**: スライド画像群 or スライド動画
- **スライド構成（3段構成を厳守）**:
  1. 「今からやることはこれです」 → 目的・ゴールの提示
  2. 「これをしたらどうなります」 → 期待される結果・効果の説明
  3. 「じゃあ実際にやります」 → 実演パートへの遷移

### Step 3: 画面操作録画
- **入力**: Step1の操作手順
- **処理**: Playwrightでブラウザを自動操作し、画面を動画として録画
- **出力**: 操作録画動画ファイル（.webm）
- **技術要件**:
  - Playwrightの `recordVideo` オプションを使用
  - 対象サービス: ChatGPT, Claude, Midjourney等
  - AIの応答完了を検知してから次の操作に進む（waitForSelector等）
  - 操作はサクサク・高速に見えるよう設計する
- **注意点**:
  - 対象サービスのUI変更に対する耐性を考慮（セレクタの柔軟な指定）
  - ログイン状態の維持（Cookie / セッション管理）

### Step 4: 出力精査（自動）
- **入力**: Step3の録画動画 + Step1の台本
- **処理**: AIが録画内容と台本の整合性を自動チェック
- **出力**: 精査結果（OK / NG + 理由）
- **チェック項目**:
  - 操作が正常に完了しているか
  - AIの出力内容が期待通りか
  - 画面が正しく表示されているか（エラー画面等の検出）
  - NGの場合 → Step3を自動リトライ

### Step 5: ナレーション合成 → 完成動画
- **入力**: Step2のスライド動画 + Step3の操作録画 + Step1のナレーション原稿
- **処理**:
  1. ナレーション原稿からAI音声を生成（ElevenLabs / VOICEVOX等）
  2. スライド動画 → 操作録画の順に結合
  3. 音声を映像に合成（タイミング調整含む）
  4. トランジション・テロップ等の追加
- **出力**: 完成動画ファイル（.mp4）

## 技術スタック（想定）

| レイヤー | ツール/技術 |
|---|---|
| 台本生成 | Claude API |
| スライド生成 | Python (python-pptx → 画像化) or HTML → Puppeteer |
| ブラウザ自動操作 | Playwright (Node.js or Python) |
| 画面録画 | Playwright recordVideo |
| 出力精査 | Claude API (Vision) でスクショ解析 |
| 音声生成 | ElevenLabs API / VOICEVOX |
| 動画編集・合成 | FFmpeg |
| オーケストレーション | Node.js or Python スクリプト |

## 実行フロー

```
[テーマ入力]
    ↓
[Step1] Claude API → 台本生成
    ↓
[Step2] 台本 → 研修スライド生成（目的→結果→実演の3段構成）
    ↓
[Step3] Playwright → ブラウザ自動操作 + 画面録画
    ↓
[Step4] Claude Vision API → 録画の自動精査
    ├── OK → Step5へ
    └── NG → Step3をリトライ
    ↓
[Step5] AI音声生成 + FFmpegで合成 → 完成動画出力
```
