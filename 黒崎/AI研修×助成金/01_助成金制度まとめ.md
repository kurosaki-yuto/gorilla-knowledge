# AI研修に使える助成金制度まとめ

> 最終更新: 2026年3月19日
> 情報ソース: 厚生労働省公式・各社労士事務所の解説記事等

## 全体像：3つの使える制度

```
┌─────────────────────────────────────────────────────┐
│           人材開発支援助成金（厚生労働省）               │
│                                                     │
│  ① 事業展開等リスキリング支援コース ← 本命            │
│     助成率75% / 賃金助成あり / 設備投資助成あり        │
│     本格AI研修向き（10時間以上）                       │
│                                                     │
│  ② 人への投資促進コース（定額制訓練）← サブスク型     │
│     助成率60% / 月2万円×12ヶ月=24万円/人              │
│     全社員に広く浅く展開するのに最適                    │
│                                                     │
│  ※ ①②ともに令和8年度（2027年3月末）で終了予定        │
├─────────────────────────────────────────────────────┤
│  ③ 東京都DXリスキリング助成金                         │
│     助成率75% / 短時間研修向き（3〜10時間）            │
│     都内中小企業限定                                  │
└─────────────────────────────────────────────────────┘
```

## ① 事業展開等リスキリング支援コース（本命）

### 概要

- 企業が **DX・GX・新規事業展開** のために従業員に訓練を実施した場合に、**経費＋賃金** を助成
- 2022年12月創設の **時限措置（2027年3月末まで）**
- AI研修はDXの中核なので **最も通りやすいカテゴリ**

### 対象企業の要件

- 雇用保険適用事業所
- 職業能力開発推進者を選任済み
- 事業展開等実施計画を作成していること
- 訓練中の従業員に賃金を適正支払いしていること

### 対象労働者の要件

- 訓練期間中に雇用保険被保険者であること
- 実訓練時間の **80%以上** を受講すること

### 助成率・助成額

#### 経費助成

| 企業区分 | 助成率 |
|---------|-------|
| 中小企業 | **75%** |
| 大企業   | **60%** |

#### 1人あたりの経費助成上限額

| 訓練時間 | 中小企業 | 大企業 |
|---------|---------|-------|
| 10〜100時間未満 | **30万円** | 20万円 |
| 100〜200時間未満 | **40万円** | 25万円 |
| 200時間以上 | **50万円** | 30万円 |

#### 賃金助成（経費とは別に上乗せ）

| 企業区分 | 助成額 |
|---------|-------|
| 中小企業 | **1,000円/時間** |
| 大企業   | **500円/時間** |

#### 上限

- 1人1訓練あたり: 1,200時間
- 事業所年間上限: **1億円**
- 同一労働者: 3回まで

### 設備投資助成（2026年3月新設）

| 項目 | 内容 |
|-----|-----|
| 助成率 | 購入費用の **50%** |
| 上限額 | 受講者数に応じて **最大150万円** |
| 条件 | 訓練で習得した知識・技能を活用し生産性向上を図る機器・設備の購入 |
| 対象例 | GPU搭載PC、AIツールライセンス、データ分析ソフト、RPA等 |

### 訓練の条件

| 要件 | 内容 |
|-----|-----|
| 訓練形式 | **OFF-JT**（業務外訓練） |
| 最低時間 | **10時間以上** |
| 実施形態 | 通学・双方向オンライン・eラーニング・通信制・定額制 すべてOK |
| 賃金助成の対象 | **通学・双方向オンラインのみ**（eラーニング等は経費助成のみ） |
| 受講率 | 80%以上が必要 |
| 計画届の期限 | 訓練開始日の **1か月前まで（厳守）** |
| 費用支払い | 全額事業主が **立替払い**（助成金は後払い） |

### 2025年〜2026年の変更点

#### 2025年4月施行

| 変更項目 | 改正前 | 改正後 |
|--------|------|------|
| 賃金助成（中小） | 960円/時間 | **1,000円/時間** |
| 賃金助成（大企業） | 480円/時間 | **500円/時間** |
| 申請認定の事前確認 | あり | **廃止**（支給申請時に一括審査） |
| 申請様式 | コース別に別々 | **3コース共通化・書類削減** |
| テレワーク制度提出 | 必要 | **不要**（自宅OFF-JT時） |

#### 2026年3月2日施行（最新・超重要）

1. **対象訓練の大幅拡充**
   - 改正前: 「事業展開に伴う配置転換のための訓練」のみ
   - 改正後: **「おおむね3年以内の人事配置計画に基づく将来の職務習得訓練」** も対象に追加
   - **配置転換が未確定でも「将来AI人材を育てたい」という計画だけでOK**
   - 例: 事務職→営業の多能工化、女性管理職育成、AI人材育成

2. **設備投資助成の新設**（上記参照）

3. **分割支給申請が可能に**
   - 長期訓練の途中段階で分割受給申請 → キャッシュフロー負担軽減

4. **申請方法の注意**
   - 2026年3月2日以降の改正対応申請は **紙申請（郵送・持参）が必要**
   - 電子申請システム（jGrants）は改正対応を整備中

### 申請の流れ

```
STEP 1: 事前準備（訓練開始の2〜3か月前）
  └ 推進者選任・計画作成・訓練機関選定・契約経緯記録

STEP 2: 計画届の提出（訓練開始1か月前まで・厳守）
  └ 管轄労働局に提出（電子申請または郵送）
  └ ※2026年3月以降の一部改正対応は紙申請のみ

STEP 3: 訓練の実施
  └ 業務命令として所定労働時間内に実施
  └ 受講記録を保存（学習ログと出勤簿の1分単位一致チェックあり）

STEP 4: 支給申請（訓練修了後2か月以内）
  └ 証明書類・経費支払い証拠等を提出

STEP 5: 審査・支給（通常1〜3か月）
```

## ② 人への投資促進コース（定額制訓練）← サブスク型

### 概要

- サブスクリプション型（定額制）の研修サービス（Udemy Business等）を使った訓練が対象
- 全社員に広く浅くAIリテラシーを展開するのに最適
- **令和8年度（2027年3月末）で終了予定**

### 助成率・助成額

| 項目 | 中小企業 | 大企業 |
|-----|---------|-------|
| 経費助成率 | **60%** | **45%** |
| 月額上限 | **2万円/人** | **2万円/人** |
| 年間上限 | **24万円/人**（2万円×12ヶ月） | **24万円/人** |
| 賃金助成 | **なし** | **なし** |
| 年度内申請回数 | 1人3回まで | 1人3回まで |
| 事業所年間上限 | 2,500万円 | 2,500万円 |

### 10時間ルール（重要）

- 支給申請時に、**対象者全員の受講時間の"合計"が10時間以上** であること
- 1人が10時間でもいいし、5人×2時間でもOK
- **ただし1時間未満の人はカウント対象外**（＝助成ももらえない）
- 「1人だけ10時間やって残り全員0時間」→ **その1人分しか助成されない**
- 各人が最低1時間以上は受講している必要がある

### 対象となるサービスの条件

- 定額制（サブスクリプション型）の研修サービスであること
- **LMS（学習管理システム）** で受講記録が取れること
- 業務時間内に実施するOFF-JT（業務外訓練）であること
- 職務に関連する知識・技能習得が目的であること
- 特定の事業主向けに作られた訓練でないこと

### 申請の流れ

```
STEP 1: 計画届の提出（サービス契約開始の1か月前まで）
STEP 2: サービス契約・訓練実施
STEP 3: 支給申請（契約期間終了後2か月以内）
STEP 4: 審査・支給
```

## ③ 東京都DXリスキリング助成金

### 概要

東京都が独自に実施する、都内中小企業向けのDX研修助成。短時間の入門研修に使いやすい。

| 項目 | 内容 |
|-----|-----|
| 助成率 | **75%**（経費） |
| 1人あたり上限 | **75,000円** |
| 企業あたり上限 | **100万円** |
| 研修時間 | **3〜10時間未満** |
| 対象 | 都内中小企業 |

### 注意

- 国の助成金（①②）との **同一研修での重複受給は不可**
- 別の研修に対してそれぞれ申請することは可能な場合あり（要社労士確認）

## 令和8年度（2026年度）のその他の動き

| 項目 | 内容 |
|-----|-----|
| 中高年齢者実習型訓練（新設） | 45歳以上対象、OFF-JT+OJT、経費助成60%、OJT実施助成10万円/人 |
| 人への投資促進コース拡充 | 長期教育訓練休暇制度等に「手当支給助成（75%）」新設 |
| 予算規模 | 令和8年度概算要求額 539億円（前年度545億円から微減） |
| 審査厳格化 | 学習ログと出勤簿の1分単位一致チェック強化 |

## 重要な注意事項

1. **制度終了期限**: ①②ともに2027年3月末で終了。延長は未定。**駆け込み需要が見込まれる**
2. **立替払い必須**: 助成金は後払い。費用を全額先払いできる資金計画が必要
3. **計画届の1日遅れで受付不可**: 期限厳守が絶対条件
4. **値引きに注意**: 教育機関から助成金額と同等の割引を受けると不支給
5. **会社都合解雇があると不支給**: 申請期間中の会社都合解雇に注意
6. **同一研修での重複受給は不可**: ①②③を同じ研修に併用はできない
