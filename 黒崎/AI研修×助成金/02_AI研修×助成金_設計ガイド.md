## この研修で使える助成金の組み合わせ戦略

```
【最強パターン: 2段構え】

STEP 1: サブスク型で全社員のAIリテラシーを底上げ
  → ② 定額制訓練（人への投資促進コース）
  → 月2万円×12ヶ月 = 24万円/人（助成率60%）
  → 何名でもOK、ハードル低い

STEP 2: 選抜メンバーに本格AI研修を実施
  → ① リスキリング支援コース
  → 経費75% + 賃金助成 + 設備投資助成
  → 1人あたり最大52万円 + 設備150万円

※ ①と②は別の研修であれば併用可能
※ 同一研修への重複受給は不可
```

## 研修設計のポイント

### A. リスキリング支援コース向け（本格研修）

#### 時間設計

```
【推奨】100〜200時間で設計
  → 中小企業: 経費上限40万円 + 賃金助成1,000円×時間
  → 大企業:   経費上限25万円 + 賃金助成500円×時間

例: 120時間の研修（中小企業の場合）
  経費助成: 研修費53万円 × 75% = 39.75万円（上限40万円以内）
  賃金助成: 120時間 × 1,000円 = 12万円
  合計: 最大約52万円/人の助成
```

最低ラインは **10時間以上**（10〜100時間未満: 経費上限は中小30万円/大企業20万円）

#### 実施形態の選び方

| 形態 | 経費助成 | 賃金助成 | 推奨度 |
|-----|---------|---------|-------|
| 対面集合研修 | ○ | ○ | ★★★ 最も確実 |
| 双方向オンライン（Zoom等） | ○ | ○ | ★★★ コスパ最強 |
| eラーニング | ○ | × | ★★ 経費のみ |
| 通信制 | ○ | × | ★ 自習型は弱い |

**結論: 賃金助成も取るなら「双方向オンライン」がベスト。**
eラーニングと組み合わせてハイブリッドにするのが現実的。

#### カリキュラム構成例（120時間・ハイブリッド）

```
■ 双方向オンライン講義（80時間）← 経費助成+賃金助成
  ├ AI基礎・概論（8時間）
  │   └ AIの仕組み、活用事例、業界動向
  ├ 生成AI活用実践（16時間）
  │   └ ChatGPT/Claude/Geminiの業務活用、社内導入設計
  ├ プロンプトエンジニアリング（16時間）
  │   └ 基本技法、業務別プロンプト設計、RAG概論
  ├ データ分析基礎（16時間）
  │   └ Excel→BI移行、ダッシュボード構築、KPI設計
  ├ AI業務自動化ワークショップ（16時間）
  │   └ RPA+AI連携、ノーコードAIツール、業務フロー改善
  └ プロジェクト発表・振り返り（8時間）
      └ 自社業務へのAI活用企画を発表

■ eラーニング自習（40時間）← 経費助成のみ
  ├ Python基礎（動画教材）
  ├ 各種AIツールハンズオン
  └ 課題提出・小テスト
```

### B. 定額制訓練向け（サブスク型・全社展開）

#### 設計のコツ

- サブスク型研修サービス（Udemy Business等）の年間契約で設計
- **LMS（学習管理システム）で受講記録が取れること** が必須条件
- 各受講者が **最低1時間以上** は受講していること（1時間未満はカウント外）
- 全対象者の **合計受講時間が10時間以上** であること

#### コスト試算例（中小企業・10名の場合）

```
サブスク年間利用料: 1人あたり月額5,000円 × 12ヶ月 = 60,000円/人
10名合計: 600,000円

助成額: 1人あたり月額上限2万円だが、実費5,000円なので全額対象
  → 5,000円 × 60% = 3,000円/月/人
  → 年間: 36,000円/人
  → 10名合計: 360,000円の助成

企業実質負担: 600,000円 - 360,000円 = 240,000円（10名で年24万円）
```

## 設備投資助成の活用（2026年3月新設）

リスキリング支援コースで研修を実施した後、**関連設備の導入費用も追加で助成される**。

| 項目 | 内容 |
|-----|-----|
| 助成率 | 購入費用の **50%** |
| 上限額 | 受講者数に応じて **最大150万円** |
| 条件 | 訓練で習得した知識・技能を活用して生産性向上を図る機器・設備 |

### AI研修と組み合わせやすい設備

- GPU搭載PC・ワークステーション
- AIツール・SaaSライセンス（年間契約）
- データ分析用ソフトウェア（Tableau, Power BI等）
- クラウドコンピューティング環境（AWS/Azure/GCP）
- RPA・業務自動化ツール

## 価格設計のヒント

### リスキリング支援コース（本格研修）

#### 中小企業向け（助成率75%）

```
研修費 53万円の場合（100〜200時間）:
  経費助成: 53万円 × 75% = 39.75万円（上限40万円以内）
  企業実質負担: 13.25万円/人
  + 賃金助成120時間 × 1,000円 = 12万円（別途企業に入る）
  → 企業の実質負担: ほぼゼロ
```

#### 大企業向け（助成率60%）

```
研修費 42万円の場合（100〜200時間）:
  経費助成: 42万円 × 60% = 25万円（上限以内）
  企業実質負担: 17万円/人
  + 賃金助成120時間 × 500円 = 6万円（別途企業に入る）
  → 企業の実質負担: 11万円/人
```

### 営業トーク用：最大助成額シミュレーション

```
【中小企業・10名受講・120時間研修 + 設備投資の場合】

① 経費助成:    40万円 × 10人 = 400万円
② 賃金助成:    12万円 × 10人 = 120万円
③ 設備投資助成: 最大150万円

合計: 最大670万円の助成

＋ サブスク型で別途全社員展開すれば
④ 定額制訓練:   24万円 × 人数分（追加）
```

**訴求ポイント**: 「助成金を使えば実質○万円で受講可能」

## 研修パッケージ開発チェックリスト

### 制度対応

- [ ] 訓練時間は10時間以上か
- [ ] OFF-JT（業務外訓練）として設計されているか
- [ ] 双方向オンラインの比率を最大化しているか（賃金助成のため）
- [ ] カリキュラムに「DX」「AI」「デジタル化」等のキーワードが含まれているか
- [ ] 「事業展開等実施計画」の記載例を企業に提供できるか
- [ ] 2026年3月改正の「人事配置計画に基づく訓練」としても申請できる設計か

### 運用体制

- [ ] 講師の経歴・資格が明示できるか
- [ ] 受講者の出席管理の仕組みがあるか（1分単位のログ対応）
- [ ] 修了要件（80%以上の受講）の管理ができるか
- [ ] LMSで受講記録を出力できるか（定額制訓練の場合）
- [ ] 企業に提出する書類テンプレートが用意されているか

### 営業・提案

- [ ] 助成金込みの価格シミュレーション資料があるか
- [ ] リスキリング支援コース / 定額制訓練 / 東京都DXの使い分けを提案できるか
- [ ] 設備投資助成との組み合わせ提案ができるか
- [ ] 社労士との連携体制があるか（申請サポート）
