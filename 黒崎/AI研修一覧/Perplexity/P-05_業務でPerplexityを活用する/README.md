## 学習目標

この動画を見終わると、以下ができるようになります：

1. **部門別のPerplexity活用パターンを3つ以上挙げられる**（営業・企画・人事・経理/法務・全社）
2. **商談前リサーチをPerplexityで実際に実行できる**（実演録画で手順を確認）
3. **Spaces・Cometを使ったチーム活用方法を理解している**

## 修了条件

- 動画を最後まで視聴完了
- 確認テスト 80%以上（5問中4問正解）
