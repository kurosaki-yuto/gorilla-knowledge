## 学習目標

この動画を見終わると、以下ができるようになります：

1. **プロンプトとは何かを説明できる**
2. **良いプロンプトの5つの要素（役割・文脈・指示・形式・制約）を使える**
3. **悪いプロンプトと良いプロンプトの違いを具体例で比較できる**
