## 学習目標

この動画を見終わると、以下ができるようになります：

1. **営業・企画・人事など各部門でのPerplexity活用パターンを3つ以上挙げられる**
2. **Perplexityを使った業務効率化の具体的な手順を実践できる**
3. **チームでPerplexityを共有活用する方法を理解している**

## 修了条件

- 確認テスト 80%以上（5問中4問正解）
