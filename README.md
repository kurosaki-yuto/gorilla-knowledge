# gorilla-knowledge

ゴリラチームのナレッジベースです。各メンバーが学んだこと・知見・Tipsなどを蓄積していきます。

## ディレクトリ構成

```
gorilla-knowledge/
├── 前田/
├── 黒崎/
├── 野田/
├── 松本/
└── 内村/
```

各メンバーは **自分のディレクトリ** にナレッジを追加していってください。

## 使い方

1. 自分の名前のディレクトリにファイルを作成・編集してナレッジを蓄積する
2. Claude Code を使って編集する場合は、**まず自分のディレクトリを参照させてから**作業を開始する

### Claude Code での作業の始め方

作業を始める前に、自分のディレクトリの内容を Claude Code に読み込ませてください。

```
# 例: 黒崎の場合
# まず自分のディレクトリを参照させる
「黒崎/ ディレクトリの中身を確認して」

# その後、ナレッジの追加・編集を依頼する
「〇〇について学んだことをまとめて」
```

こうすることで、既存のナレッジと重複しない形で新しい知見を追加できます。
