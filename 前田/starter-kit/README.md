# スターターキット — 新メンバー向け環境構築ガイド

## 概要

このガイドに従えば、新メンバーが **30分で開発環境を構築** し、即座にビジネスタスクを開始できる状態になります。

Claude Codeにこのフォルダを読み込ませれば、セットアップを自動支援してくれます。

---

## セットアップの全体像

```
Step 1: 前提条件の確認（5分）
    ↓
Step 2: Claude Code のインストールと初期設定（10分）
    ↓
Step 3: OpenClaw Gateway の導入（10分）
    ↓
Step 4: Paperclip（タスク管理）の導入（5分）
    ↓
Step 5: Discord Bot の接続（5分）
    ↓
Step 6: 動作確認（5分）
    ↓
🎉 完了！ビジネスタスク開始可能
```

## Claude Codeへの指示

```
「starter-kit/ フォルダの中身を読んで、環境構築を手伝って」
```

この一言で、Claude Codeがこのガイドに沿ってセットアップを支援してくれます。
