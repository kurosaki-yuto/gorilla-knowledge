# 講座インポート手順

## 前提条件

- Node.js がインストール済み
- `skillplus-admin/` ディレクトリにいること
- YouTube Data API v3 が有効な Google Cloud プロジェクト
- OAuth 2.0 クライアントID が作成済み（リダイレクトURI: `http://localhost:3002/api/youtube/callback`）

## フォルダ構成（gorilla-knowledge）

```
黒崎/AI研修一覧/
  AI-101_AIの基本/
    ├── README.md      ← 講座メタデータ・学習目標
    ├── 台本.md        ← ナレーション台本（動画要約に使用）
    ├── テスト.md      ← 確認テスト（5問・4択）
    ├── 動画.mp4       ← 講座動画
    ├── スライド.pptx   ← スライド資料
    └── スライド画像/   ← スライド画像
```

## インポートされる内容

| ソース | 登録先 | 内容 |
|--------|--------|------|
| README.md | contentJson (above) | 「この講座について」「学習目標」 |
| 動画.mp4 | YouTube (限定公開) → courses.video_url | 動画 |
| 台本.md | contentJson (below) | 「動画の要約」（補足資料として折りたたみ表示） |
| テスト.md | quizzes テーブル | 確認テスト問題（正解は受講生に非表示） |

## データ構造ルール

```
研修（例: AI研修基礎編）
  └─ カテゴリー（例: 基礎講座）← グループ単位で1つ
       ├─ 講座1: AI-101: AIの基本
       ├─ 講座2: AI-102: AIツール活用入門
       └─ ...
```

- カテゴリーは講座ごとに作らない。グループ（シリーズ）単位で1つ
- 所要時間 = 動画時間 × 1.5倍（補足資料・テスト含む）

## 一括インポート

### 1. スクリプト実行

```bash
cd /Users/matsumotoshuntasuku/AI研修講座/skillplus-admin
node scripts/import-ai-courses.mjs
```

### 2. YouTube認証（初回のみ）

- ブラウザが開く → Googleアカウントでログイン → YouTube APIを許可
- 認証は `http://localhost:3002/api/youtube/callback` で受け取る（管理画面サーバーが起動している必要あり）
- トークンは `/tmp/yt-cli-token.json` に保存され、2回目以降は自動

### 3. 処理内容

各フォルダに対して以下を自動実行：

1. カテゴリー作成（既存ならスキップ）
2. `動画.mp4` → YouTube限定公開アップロード → URL取得
3. `README.md` → 学習目標を抽出 → contentJson.above に設定
4. `台本.md` → ナレーション全文を抽出 → contentJson.below に設定
5. `テスト.md` → 問題をパース → quizzes テーブルに登録
6. 動画時間を台本の「総尺」から設定
7. 所要時間 = 動画時間 × 1.5

## 単体アップロード

```bash
# 1本だけアップロード + Supabase登録
node scripts/upload-video.mjs <MP4ファイル> [講座名] [カテゴリーID]

# 例
node scripts/upload-video.mjs ~/Desktop/lecture.mp4 "新しい講座"

# カテゴリー一覧を表示
node scripts/upload-video.mjs --list-categories
```

## テスト.md のフォーマット

2つのフォーマットに対応：

### フォーマット A（details形式）

```markdown
## 問題 1

**問題文がここに入る**

- A. 選択肢1
- B. 選択肢2
- C. 選択肢3
- D. 選択肢4

<details>
<summary>正解</summary>

**B. 正解の選択肢**

解説: ...
</details>
```

### フォーマット B（シンプル形式）

```markdown
## Q1. 問題文がここに入る

- A. 選択肢1
- B. 選択肢2
- C. 選択肢3
- D. 選択肢4

**正解: B**
> 解説: ...
```

## 環境変数（.env.local）

```
YOUTUBE_CLIENT_ID=<Google Cloud OAuth クライアントID>
YOUTUBE_CLIENT_SECRET=<クライアントシークレット>
YOUTUBE_REDIRECT_URI=http://localhost:3002/api/youtube/callback
```

## トラブルシューティング

### redirect_uri_mismatch
Google Cloud Console → 認証情報 → OAuth クライアントID → 承認済みリダイレクトURI に以下を追加：
- `http://localhost:3002/api/youtube/callback`

### テストが0問で登録される
テスト.md のフォーマットが上記のどちらにも当てはまらない可能性。フォーマットを確認して調整。

### 動画時間が0分
台本.md に `総尺: 約10分` の記載がない場合、デフォルト10分になる。管理画面から手動で修正可能。
